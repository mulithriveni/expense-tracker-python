# Expense Tracker

## Description
Expense Tracker is a simple Python application that helps users record and manage daily expenses. Users can add expenses, view all expenses, and calculate total spending.

## Features
- Add expenses
- View expense history
- Calculate total expenses
- Easy command-line interface

## Technologies Used
- Python

## How to Run
1. Open terminal
2. Run:
   python expense_tracker.py
